Podstawy tworzenia stron internetowych

Witamy w drugim module kursu Code Brainers "Fullstack Developer". Na kolejnych spotkaniach nauczycie się podstaw tworzenia stron internetowych przy pomocy języków HTML, CSS i JavaScript. Dowiecie się, jak działa Internet i popularne strony internetowe, poznacie sposoby upiększania stron oraz metody uczynienia ich bardziej interaktywnymi. Dział ten będzie się różnił od pozostałych, ale równocześnie pokażemy Wam narzędzia, dzięki którym będziecie mogli łatwo przygotować ciekawe i estetyczne strony internetowe.
